# Obys Agency UI Clone

A visually immersive **UI clone of the Obys Agency website**, built to practice advanced **frontend animations, smooth scrolling, and modern web design techniques**.

This project focuses on recreating the **interactive and cinematic user experience** of the original Obys website using modern JavaScript animation libraries.


## Project Overview

The goal of this project was to understand how modern creative agencies build **highly interactive landing pages** with smooth animations and fluid user interactions.

This clone replicates:

- Smooth scrolling effects
- GSAP powered animations
- Scroll-triggered transitions
- Interactive image effects
- Modern typography and layout

The project helped in gaining hands-on experience with **animation-based UI development**, which is commonly used in **creative portfolios and agency websites**.

---

## Technologies Used

- **HTML5** – Structure of the webpage  
- **CSS3** – Styling and layout  
- **JavaScript (ES6)** – Interactivity and animation control  

### Libraries Used

- **GSAP (GreenSock Animation Platform)** – High-performance animations  
- **ScrollTrigger** – Scroll-based animation control  
- **Locomotive Scroll** – Smooth scrolling experience  
- **Shery.js** – Image distortion and visual effects  

---

## Features

- Smooth scrolling experience
- Scroll-triggered animations
- Modern creative agency UI
- Interactive image distortion effects
- Clean responsive layout
- Animated text reveals
